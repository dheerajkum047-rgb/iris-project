# 🌸 Iris Dataset - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Iris dataset to understand the structure, relationships, and patterns within the data. The analysis includes univariate, bivariate, and multivariate visualizations using Python.

---

## 📊 Dataset

* Dataset Used: Iris Dataset
* Total Records: 150 (or your dataset size)
* Features:

  * Sepal Length
  * Sepal Width
  * Petal Length
  * Petal Width
* Target Variable:

  * Species (Setosa, Versicolor, Virginica)

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 🔍 Exploratory Data Analysis Steps

### 1. Data Preparation

* Loaded dataset using Pandas
* Checked structure using `info()` and `describe()`
* Verified missing values

### 2. Univariate Analysis

* Histograms for distribution
* Boxplots for outlier detection
* Countplot for species distribution

### 3. Bivariate Analysis

* Scatter plots to analyze relationships
* Bar plots for comparison

### 4. Correlation Analysis

* Correlation matrix
* Heatmap visualization

### 5. Pairplot Analysis

* Visualized relationships between all features

---

## 📈 Key Insights

* Petal length and petal width show strong correlation
* These features are most important for classification
* Setosa species is clearly separable
* Versicolor and Virginica show slight overlap
* Dataset is clean with no missing values
